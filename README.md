# FastAPI 할일목록 웹 개발

## 사용한 기술
- fastapi: 백엔드
- jinja2: 프론트엔드
- pydantic: 모델 검증
- uvicorn: 서버

## 설명
- FastAPI 애플리케이션을 개발하기 위한 개발환경 구축을 학습(docker, venv, github)
- 기본적인 CRUD 학습
- jinja2 템플릿을 사용한 뷰 생성 방법과 API 응답 렌더링 방법 학습
- 응답 모델, 오류 처리, 상태 코드 학습
